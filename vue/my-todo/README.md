# my-todo

## 사용 기술
* vue-cli 3.0
* vuex

## Project Setup
### Project Create
```
vue create my-todo
cd my-todo
vue i vuex
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
